## net_onlineRepresentatives
Return online representative accounts that have voted recently  
- **Parameters**: `null`
  
- **Returns**: 
  - `[]address`:  addresses list for representative accounts 

- **Example**:

::: demo
```json tab:Request
{
  "jsonrpc": "2.0",
  "id":2,
  "method":"net_onlineRepresentatives"
}
```
