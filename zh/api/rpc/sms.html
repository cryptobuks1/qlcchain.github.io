# sms_phoneBlocks

# sms_messageHash

  • Parameters:

    • string : 短信

  • Returns:

    • string : 短信哈希

  • Example:

	"id": 1,
	"method": "sms_messageHash",
	"params": [
		"qlcqlc"
	]
}

  "result": "bfa9a2661b022debbe0de95c89d08c6682928bb752eec0ed712b0aca2733aab7"

	]
}
