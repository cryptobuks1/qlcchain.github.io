# rewards_getTotalRewards

根据抵押的交易 ID 查询获得的空投 gas 总量

  • Parameters:

    • string : 抵押的交易 ID

  • Returns:

    • uint64 : 获得的空投总量

  • Example:

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "rewards_getTotalRewards",
	"params": [
        "04223e8259cf00b84cfe31ef895073a4154ac88c66c43efa977b9c8c714fbeda"
	]
}

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": 14000000000
}

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "rewards_getTotalRewards",
	"params": [
	]
}

# rewards_getConfidantRewards

根据 confidant 地址查询 confidant 的空投收益

  • Parameters:

    • string : confidant 地址

  • Returns:

    • map : 空投收益 - key : confidant id 的 hash 值 - value : 收益量

  • Example:

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "rewards_getConfidantRewards",
	"params": [
		"qlc_3947hepb6ipq1m8b1jdbi6h7te3epqiakeb1j59ppwmi7wnj5optopsdgo5g"
	]
}

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "6ba9a8dff67863b72a77b07fce4e0ba08da9b36c662bdba4371e784e882c6e73": 2742934081058
  }
}

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "rewards_getConfidantRewards",
	"params": [
		"qlc_3947hepb6ipq1m8b1jdbi6h7te3epqiakeb1j59ppwmi7wnj5optopsdgo5g"
	]
