# contract_packContractData

根据ABI打包指定的方法

  • Parameters:

    • string: ABI数据
    • string: 方法名称
    • []string: 方法参数

  • Returns:

    • string: 打包结果

  • Example:

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "contract_packContractData",
	"params": [
		"[{ \"type\" : \"function\", \"name\" : \"balance\", \"constant\" : true }]",
		"balance",
		[]
	]
