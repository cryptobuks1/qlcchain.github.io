# contract_packContractData

Pack the given method name to conform the ABI

  • Parameters:

    • string : abi string
    • string : method name
    • []string : arguments for the method

  • Returns:

    • string : packed result

  • Example:

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "contract_packContractData",
	"params": [
		"[{ \"type\" : \"function\", \"name\" : \"balance\", \"constant\" : true }]",
		"balance",
		[]
	]
}
