# sms_messageHash

Return hash for message

  • Parameters:

    • string : message

  • Returns:

    • string : hash for the message

  • Example:

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "sms_messageHash",
	"params": [
		"qlcqlc"
	]
}

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "bfa9a2661b022debbe0de95c89d08c6682928bb752eec0ed712b0aca2733aab7"
}

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "sms_messageHash",
	"params": [
		"qlcqlc"
	]
}

# sms_messageStore

Store message and return message hash

  • Parameters:

    • string : message

  • Returns:

    • bool : hash for the message

  • Example:

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "sms_messageStore",
	"params": [
		"qlcqlc"
	]
}

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "bfa9a2661b022debbe0de95c89d08c6682928bb752eec0ed712b0aca2733aab7"
}

{
	"jsonrpc": "2.0",
	"id": 1,
	"method": "sms_messageStore",
	"params": [
		"qlcqlc"
	]
}
