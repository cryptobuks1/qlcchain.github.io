# util_rawToBalance

Return balance by specific unit for raw value

  • Parameters:

    • string: raw value
    • string: unit
    • string: optional, token name , if not set , default is QLC

  • Returns:

    • string: balance for the unit

  • Example:

{
  "jsonrpc": "2.0",
  "id":1,
  "method":"util_rawToBalance",
  "params":  ["200000000","QLC"]

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "2"

{
  "jsonrpc": "2.0",
  "id":1,
  "method":"util_rawToBalance",
  "params":  ["200000000","QLC"]

# util_balanceToRaw

Return raw value for the balance by specific unit

  • Parameters:

    • string : balance
    • string : unit
    • string : optional, token name, if not set , default is QLC

  • Returns:

    • string: raw value

  • Example:

{
  "jsonrpc": "2.0",
  "id":1,
  "method":"util_balanceToRaw",
  "params":  ["2","QLC"]

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "200000000"

{
  "jsonrpc": "2.0",
  "id":1,
  "method":"util_balanceToRaw",
  "params":  ["2","QLC"]
