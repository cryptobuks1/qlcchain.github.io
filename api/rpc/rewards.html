# rewards_getTotalRewards

returns total airdrop qgas amount for a specific pledge

  • Parameters:

    • string: transaction id for the pledge

  • Returns:

    • uint64: total rewards

  • Example:

{
	"jsonrpc": "2.0",
	"id": 1,
