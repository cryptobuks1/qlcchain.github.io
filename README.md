## QLC Chain docs

[![Build Status](https://travis-ci.org/qlcchain/qlcchain.github.io.svg?branch=raw)](https://travis-ci.org/qlcchain/qlcchain.github.io)

### Env

- node: >= 9.xx
- yarn: >= 1.3.xx

### Install vuepress

```
yarn global add vuepress@next
```

### Start Writing

```
yarn docs:dev
```

### build

```
yarn docs:build
```
